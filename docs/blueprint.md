# **App Name**: FWstudio Digital Transformation

## Core Features:

- Free Consultation Booking: Allow potential clients to book a free consultation through a form. Include fields for name, company, email, and a brief message.
- Project Showcase: Display past projects with descriptions, technologies used, and results achieved.
- Solutions Overview: Clearly outline the problems FWstudio solves and the process they follow to help businesses.
- Lead Capture & Management: Collect leads from the contact form and store them for follow-up (consider integration with a CRM).
- Personalized Solution Recommendation: Use a generative AI tool that analyzes user input (company details, challenges) and provides personalized digital transformation recommendations, emphasizing automation opportunities. The LLM will have tools to include particular services offered by FWstudio when it judges them most appropriate.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to convey trust, stability, and professionalism, fitting for a business solutions provider.
- Background color: Light Gray (#FFFFFF) for a clean and modern look, ensuring readability and contrast.
- Accent color: Teal (#60A5FA) to highlight CTAs and important information, offering a fresh and modern touch.
- Headline font: 'Poppins', a geometric sans-serif for a precise and fashionable feel.
- Body font: 'Inter', a humanist sans-serif for readability and a touch of warmth. This will be used in conjunction with Poppins for headlines.
- Use clean, minimalist icons to represent services and solutions. Icons should be in the teal accent color.
- Employ a grid-based layout for a structured and responsive design. Use white space effectively to avoid a cluttered look.
